About this site

這個網站怎麼更新

  1. 你貼資料：新聞、文章、截圖、連結或自己的筆記。
  2. 我補來源：查原文與相關資料，但不整篇搬運。
  3. 寫入資料庫：在 data/articles/ 新增一個 JSON 檔，加進 index.json 順序清單。
  4. 建置：跑 node scripts/build-articles.mjs，自動產生 data/articles.js 與 sitemap.xml。
  5. 部署更新：GitHub 更新後 Cloudflare Pages 自動發布。
